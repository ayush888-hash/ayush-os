# Ayush OS — Personal Execution System

Your personal 3-year placement preparation tracker.

## Setup (one time)

1. Install Node.js from https://nodejs.org (LTS version)
2. Open terminal in this folder
3. Run: `npm install`
4. Run: `npm run dev`
5. Open http://localhost:5173 in your browser
6. PIN: 2342

## Host on GitHub Pages (free, works on mobile)

1. Push this folder to a GitHub repo
2. Run: `npm run build`
3. Install gh-pages: `npm install gh-pages --save-dev`
4. Add to package.json scripts: `"deploy": "gh-pages -d dist"`
5. Run: `npm run deploy`
6. Go to repo Settings → Pages → Source: gh-pages branch
7. Your site will be live at: https://yourusername.github.io/repo-name
